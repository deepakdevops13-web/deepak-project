This repository demonstrates a complete end-to-end DevOps workflow for deploying and managing a cloud-native application using modern DevOps tools and best practices. It is designed as a real-world DevOps project showcasing how production systems are built, automated, and scaled.

Key features include:

Infrastructure provisioning with Terraform

Containerization using Docker

Orchestration with Kubernetes

CI/CD automation via Jenkins pipelines

Configuration management with Ansible & Vagrant

Secure secrets management

Scalable cloud architecture on AWS
